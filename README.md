# Advanced Web Technologies (CSC 4161)

This repository serves as a central hub for all laboratory exercises, assignments, and project work completed as part of the **Advanced Web Technologies** course.

## 📚 Course Overview
The course covers modern web development practices, focusing on scalable backend architectures, frontend frameworks, and full-stack integration.
